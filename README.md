# 💊 Medicine Reminder

A simple Python application that helps users keep track of their medicine schedule.

## Features

- Add medicine reminders
- View all reminders
- Delete reminders
- Easy menu-driven interface

## Technologies Used

- Python 3

## How to Run

1. Clone the repository.

2. Open the project folder.

3. Run the program:

```bash
python medicine_reminder.py
```

## Sample Menu

```
===== Medicine Reminder =====

1. Add Medicine
2. View Medicines
3. Delete Medicine
4. Exit
```

## Example

```
Enter Medicine Name:
Paracetamol

Enter Time:
8:00 AM

Medicine reminder added successfully!
```
